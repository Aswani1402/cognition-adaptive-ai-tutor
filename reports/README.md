# Reports

Use this folder for selected lightweight evidence reports and charts.

Recommended contents:

- Final backend evaluation summary
- Frontend build and feature verification summary
- CogniTutorLM evaluation summary
- Fine-tuning comparison summary
- Developer demo audit summary
- A small number of reviewer-friendly charts

Do not bulk-copy raw `outputs/`, `evaluation_outputs/`, checkpoints, datasets, private mappings, or large archives into this folder.
